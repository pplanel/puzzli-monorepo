# generate-color

Generates the custom properties for the theme based on the $colors map.

```scss
@mixin generate-custom-properties($name, $weights) {
  @each $key, $value in $weights {
    --app-color-#{""+$name}-#{""+$key}-rgba: #{list-color-rgb($value)};
    --app-color-#{""+$name}-#{""+$key}: #{$value};
  }
}
```

## Parameters

| Parameter | Description                          | Type                                                                                                                            | Default |
| --------- | ------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------- | ------- |
| `name`    | The color present in the colors map. | `"primary"` \| `"secondary"` \| `"tertiary"` \| `"success"` \| `"warning"` \| `"danger"` \| `"light"` \| `"medium"` \| `"dark"` | `-`     |
| `weights` | The color weights in the colors map. | `number`                                                                                                                        | `-`     |

## Usage

```scss
$colors: (
  primary: (
    100: #fdf4f5,
    200: #f5bfc5,
    300: #ed8a96,
    400: #e65469,
    500: #df1d3d,
    600: #b51433,
    700: #8a0d28,
    800: #5e071c,
    900: #30030f
  )
);

:root {
  @each $key, $value in $colors {
    @include generate-custom-properties($key, $value);
  }
}
```
