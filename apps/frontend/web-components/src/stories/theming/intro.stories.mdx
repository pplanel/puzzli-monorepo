# Intro

You may apply diferent theme's based on a class added to the document body or use the default theme.

## Import Theme Styles

```ts
// Add this import to your JS application root
import 'prodigio-components/dist/themes/<theme_name>.css';
```

```html
<!-- or add the style tag in your html page -->
<link id="theme" href="prodigio-components/dist/themes/<theme_name>.css" rel="stylesheet" />
```

```html
<!-- and add the related class to the page body -->
<body class="<theme_name>"></body>
```
